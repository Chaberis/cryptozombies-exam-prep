# Solidity: Beginner to Intermediate Smart Contracts

Пакет материалов по первому треку CryptoZombies (уроки 1–6), обновлённый под подготовку к экзамену: не просто карта уроков, а краткие summaries, ключевые паттерны и код, который нужно понимать.

## Что внутри

- `lesson-1.md` … `lesson-6.md` — экзамен-ориентированные конспекты по каждому уроку.
- `theory.md` — общий теоретический конспект по всему треку: термины, понятия, архитектура, ERC721, SafeMath, Web3.js.
- `lessons-1-6.json` — структурированная карта уроков и глав.

## Как читать пакет

1. Сначала читай `summary` и блок `Что нужно уметь после урока`.
2. Потом разбирай секцию `Код, который нужно понимать` и проговаривай, зачем нужен каждый фрагмент.
3. Закрывай урок чеклистом и типовыми заданиями — это ближайший эквивалент экзаменационной подготовки.

## Ограничения источников

- Уроки `1–5` имеют RU-локаль в `cryptozombie-lessons`.
- Урок `6` есть только в английском source в репозитории.
- В `cryptozombies-lesson-code` лежат chapter snapshots, а не единый runnable project.
