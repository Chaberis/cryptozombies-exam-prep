# Beyond Ethereum: Explore the Blockchain Ecosystem

Пакет материалов по треку CryptoZombies `Beyond Ethereum: Explore the Blockchain Ecosystem`.

## Что внутри

- `lesson-17.md` — Intro to zkSync.
- `lesson-18.md` — Advanced zkSync Concepts.
- `theory.md` — общий теоретический конспект по zkSync, ZK-rollups и multi-token workflow.
- `lessons-17-18.json` — структурированные данные по урокам.

## Как читать пакет

1. Сначала прочитать `theory.md`, чтобы собрать общую модель zkSync и ZK-rollup архитектуры.
2. Затем пройти `lesson-17.md`, потому что lesson 18 прямо опирается на него.
3. После этого пройти `lesson-18.md`.
4. При необходимости открывать исходные markdown-файлы из `en/17` и `en/18`.

## Ограничения источников

- В `cryptozombie-lessons` для lessons `17` и `18` нет русской локали.
- В `cryptozombies-lesson-code` нет отдельных каталогов `lesson-17/` и `lesson-18/`.
- Для lessons `17` и `18` код нужно брать из `answer` блоков самих markdown-глав `en/17/*` и `en/18/*`.
