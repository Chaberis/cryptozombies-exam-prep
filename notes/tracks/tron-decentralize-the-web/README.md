# Tron: Decentralize the web

Пакет материалов по треку CryptoZombies `Tron: Decentralize the web`.

## Что внутри

- `lesson-20.md` — Deploying to TRON.
- `theory.md` — общий теоретический конспект по TRON, TronBox, TronWeb и deploy workflow.
- `lessons-20.json` — структурированные данные по уроку.

## Как читать пакет

1. Сначала прочитать `theory.md`, чтобы собрать общую модель экосистемы TRON и deployment flow.
2. Затем пройти `lesson-20.md`.
3. После этого при необходимости открывать исходные markdown-файлы из `en/20`.

## Ограничения источников

- В `cryptozombie-lessons` для lesson `20` нет русской локали.
- В `cryptozombies-lesson-code` нет каталога `lesson-20/`.
- Для lesson `20` код нужно брать из `answer` блоков markdown-глав `en/20/*`.
