# Advanced Solidity: Get In-depth Knowledge

Пакет материалов по треку CryptoZombies `Advanced Solidity: Get In-depth Knowledge`.

## Что внутри

- `lesson-10.md` — Deploying DApps with Truffle.
- `lesson-11.md` — Testing Smart Contracts with Truffle.
- `lesson-14.md` — How to Build an Oracle.
- `lesson-15.md` — How to Build an Oracle - Part 2.
- `lesson-16.md` — How to Build an Oracle - Part 3.
- `theory.md` — общий теоретический конспект по Truffle, тестированию и oracle architecture.
- `lessons-10-16.json` — структурированные данные по урокам.

## Как читать пакет

1. Сначала прочитать `theory.md`, чтобы собрать общую модель трека.
2. Затем пройти уроки по порядку: `10 -> 11 -> 14 -> 15 -> 16`.
3. После этого при необходимости открывать исходные markdown-файлы из `en/10`, `en/11`, `en/14`, `en/15`, `en/16`.

## Ограничения источников

- В `cryptozombie-lessons` для lessons `10, 11, 14, 15, 16` нет русской локали.
- В `cryptozombies-lesson-code` есть материал только для `lesson-10` и `lesson-11`, и он неполный.
- Для `lesson-14`, `lesson-15`, `lesson-16` код нужно брать из `answer` блоков самих markdown-глав `en/14/*`, `en/15/*`, `en/16/*`.
- Уроки `12` и `13` отсутствуют и на витрине курса, и в этом репозитории. Трек `10-16` фактически состоит из `10, 11, 14, 15, 16`.
