# Chainlink: Decentralized Oracles

Пакет материалов по треку CryptoZombies `Chainlink: Decentralized Oracles`, урок `19`.

## Что внутри

- `lesson-19.md` — экзамен-ориентированный конспект по уроку.
- `theory.md` — общий теоретический конспект по теме оракулов, data feeds и VRF.
- `lessons-19.json` — структурированные данные по уроку и главам.

## Как читать пакет

1. Сначала прочитать `theory.md`, чтобы собрать общую модель оракулов.
2. Затем пройти `lesson-19.md`: summary, ключевые концепции, код и типовые задания.
3. После этого при необходимости открывать исходные markdown-файлы из `en/19/*`.

## Ограничения источников

- В `cryptozombie-lessons` для lesson 19 нет русской локали.
- В `cryptozombies-lesson-code` нет отдельного репозитория sample code для lesson 19, поэтому код взят из `answer` блоков самих markdown-глав.
- На сайте lesson 19 имеет `Total Chapters 12`, а в `en/index.ts` / каталоге `en/19` видно 13 элементов, потому что в репо отдельно лежит `00-overview.md`, который сайт, [INFERENCE], не считает отдельной главой.
